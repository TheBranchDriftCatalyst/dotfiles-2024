## ğŸ’¡ Pro Tips for Beginners
1. Use 'jk' to exit insert mode (faster than Esc)
2. Practice hjkl movement instead of arrow keys
3. Use <leader>ch to open this cheatsheet anytime
4. <leader>? shows you all available keymaps
5. Start with basic movements, add complexity gradually
6. Use :Tutor for built-in vim tutorial
7. Flash plugin: Press 's' + 2 characters to jump anywhere
8. Which-Key will show you options after pressing <leader>

## ğŸ“ Learning Path
1. Master basic movement (hjkl, w, b, 0, $)

3. Practice copy/paste (yy, dd, p, P)
4. Use file finder (<leader>ff)
5. Learn visual mode (v, V, Ctrl+v)
6. Master search and replace
7. Explore plugins gradually

## âš ï¸ Known Issues & Fixes

### Command Mode (`:`) Issues
If pressing `:` causes screen blanks or crashes:
- Treesitter is disabled to fix directive conflicts with Neovim 0.11.4
- Syntax highlighting still works via built-in Vim highlighting
- All other features work normally

### Plugin Loading Issues
If plugins fail to load:
- Run: `:Lazy sync` to update all plugins
- Or restart nvim to auto-install missing plugins
